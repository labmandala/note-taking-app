<h1>Interactive Note Taking App</h1> 
<h4>A greenfield project built with React where users create & organize their notes.</h4>

<p>The note app UI lets users:</p>
<li>
<ul>* Create new sticky notes</ul>
<ul>* Edit notes</ul>
<ul>* Delete notes</ul>
<ul>* Search notes</ul>
<ul>* Organize to-do items </ul>
</li>
